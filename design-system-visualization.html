6. Priority Action Items

High Priority

  • ✅ Standardize theme naming: Migrate cBSDigitalCTV and entertainmentWeb to light/dark
  • ✅ Unify overlay property naming: Critical inconsistency affecting 5+ brands
  • ✅ Add focus states: Only 3/12 brands have them (accessibility requirement)
  • ✅ Complete button state coverage: Add hover/focus/disabled/loading where missing
  • ✅ Standardize elevation system: Choose one approach (numbered vs semantic)

Medium Priority

  • ⚠️ Add dark theme to entertainmentWeb
  • ⚠️ Remove "tokens" wrapper from fantasyApp
  • ⚠️ Standardize table properties across all brands
  • ⚠️ Standardize text link naming and add missing states

Low Priority

  • 📝 Document brand-specific extensions (fantasy gradients, SportsLine odds)
  • 📝 Create base color palette from common values
  • 📝 Fix capitalization inconsistencies (brandSportsline vs brandSportsLine)

Summary

This visualization shows that while most brands follow similar patterns, there are critical gaps in:

  • Accessibility: Only 25% of brands have focus states
  • Consistency: Overlay naming has 5+ different patterns
  • Completeness: Several brands missing dark themes, elevation systems, and button states

Recommendation: Prioritize focus states (accessibility) and overlay naming unification (consistency) as these affect all users and developers.
